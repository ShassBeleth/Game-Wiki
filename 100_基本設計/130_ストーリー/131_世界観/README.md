2X32年ーー。地球温暖化の進行に伴い海抜は上昇。突如海に沈みバラバラとなった日本諸島は未だ統一できずにいた。  
2X38年8月。制圧部隊による反社会組織への武力行使が行われる。  
同年12月。幾千もの反社会組織を制圧してきた坂城の犠牲の元、日本最大のテロ組織リーダー玖瀬 薫を拘束。未だ戦火は絶えないが、確実に日本は平和へと近づいていた。  
2X39年5月。反社会組織への武力行使にドローンが初めて用いられる。その後、AIの精度の向上とともに制圧率は上がったが、突発的な問題に対応できず、ドローンだけでは被害を最小限に抑えることができなかった。しかし、制圧部隊の犠牲は少なく済むことが評価されるようになった。  
2X42年11月。拘束されていた玖瀬 薫の死刑執行に伴う移送中にテロが発生。混乱に乗じて玖瀬 薫が脱走してしまう。後に、この事件は玖瀬 薫率いるテロ組織の残党による玖瀬奪還作戦だったことがわかる。  
そして2X43年1月ーー。ドローンへのリモートアクセスを可能とさせる開発案が提出される。複数のドローンへのアクセスを瞬時に切り替え、制圧部隊の人員を最小限に抑えながらも、何倍もの制圧力を持つドローン部隊が結成されると予想された。  
このプロジェクトの開発メンバー水橋と深山の技術力により予定の半分の時間で開発は終了しようとしていたが・・・。  

# 備考
ドローンへのリモートアクセスが狙われる  
ドローンだけだと結構雑魚  
組織にはすでにリモート操作のエキスパートがいる  
ドローンへのアクセス権が奪えれば組織のメンバーにドローンが使える
